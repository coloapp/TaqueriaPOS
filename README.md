# TaqueriaPOS

Sistema de gestión para taquería offline.

## Inicio rápido
Para que el asistente entre en contexto rápidamente, puedes pedirle que lea el archivo `load_context.js` o que revise la memoria en `.gemini/tmp/taqueriapos/memory/`.

## Estructura
- `/www`: Archivos web (HTML, CSS, JS).
- `/src`: Lógica de la aplicación.
- `/sql`: Scripts de base de datos.
