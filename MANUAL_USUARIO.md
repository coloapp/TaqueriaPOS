# Manual de Usuario - TaqueriaPOS

¡Bienvenido a **TaqueriaPOS**! Este sistema está diseñado para agilizar el servicio en tu taquería, permitiendo que meseros, taqueros y cajeras trabajen en sincronía.

## 1. Inicio, Activación y Guía
Al abrir la app por primera vez en el dispositivo principal (Caja):
1. Ingresa el nombre de tu taquería, teléfono y dirección.
2. Verás un **ID de Dispositivo** (ej: `TPOS-X1Y2Z3`).
3. Ingresa el código de activación (es tu ID escrito al revés).
4. El sistema se desbloqueará y guardará tu configuración permanentemente.
5. **Mini-Tour:** Una vez activada, la app te dará una breve guía interactiva para conocer las secciones principales.

## 2. Inventario Inicial
El sistema incluye un catálogo precargado para que empieces a trabajar de inmediato:
*   **Tacos:** Pastor, Suadero, Bistec, Tripa, Cabeza, Chorizo.
*   **Especiales:** Quesadillas, Lonches, Volcanes, Papa Rellena.
*   **Bebidas:** Aguas (1L y 0.5L), Refrescos.
*   **Postres:** Flan.
*   **Dulces:** Mazapán, Paleta Payaso.

*Nota: Puedes editar precios y productos en el menú lateral -> Catálogo de Productos.*

## 3. Flujo de Trabajo (Mesero)
1. Entra a la sección **Mesas**.
2. Selecciona una mesa libre (color verde).
3. Selecciona los productos de las pestañas superiores.
    *   **Notas Rápidas:** Usa los botones "S/ Cebolla" o "S/ Cilantro" para platos específicos.
    *   **Platos:** Puedes separar la orden por platos individuales usando el botón "+ Plato".
4. Presiona **ENVIAR PEDIDO**. 
    *   Esto imprimirá automáticamente el ticket de cocina con las instrucciones para el taquero.
    *   La mesa cambiará a color rojo (ocupada).

## 4. Agregar "Extras" y Pedir Cuenta (Mesero)
Si los clientes piden más cosas después de su primer pedido:
1. Selecciona la mesa ocupada (roja).
2. Agrega los nuevos productos y presiona **AGREGAR EXTRAS**.
    *   Los productos se suman automáticamente a la cuenta.
3. **Pedir Cuenta:** Cuando el cliente esté listo para pagar, el mesero puede presionar **📄 CUENTA** desde su tablet.
    *   Esto imprimirá el ticket de cuenta para el cliente.
    *   La mesa cambiará a color **amarillo** en el mapa, indicando que el cliente está por pagar.
    *   *Importante:* La mesa sigue ocupada y no se liberará hasta que el cajero confirme el pago.

## 5. Monitor de Cocina (Taquero)
El taquero tiene una vista dedicada llamada **Monitor**:
*   Verá los pedidos pendientes en tarjetas.
*   Cada tarjeta muestra qué mesa es, qué pidió y las notas (sin cebolla, etc.).
*   Puede marcar pedidos como "Entregados" para limpiar su pantalla.
*   Puede reimprimir una orden si el papel se dañó.

## 6. Cobro y Métodos de Pago (Caja)
Cuando el cliente pase a pagar a la caja:
1. Ve a la sección **Caja**.
2. Las mesas que ya pidieron cuenta aparecerán con un aviso de **🔔 CUENTA** (color amarillo).
3. Presiona **COBRAR**.
4. Selecciona el método de pago:
    *   **💵 Efectivo**
    *   **📱 Transferencia**
    *   **💳 Tarjeta / Terminal**
5. Una vez seleccionado, la mesa se liberará (color blanco) y el dinero se registrará en el reporte de ventas según el método elegido.

## 7. Corte de Caja
Al final de la jornada:
1. Ve a **Caja** -> **REALIZAR CORTE**.
2. Ingresa el monto real que tienes en efectivo.
3. El sistema generará un informe comparando lo vendido vs. lo contado, mostrando si falta o sobra dinero.
4. Se generará un PDF que puedes compartir o imprimir.

## 8. Configuraciones Avanzadas y Catálogo
En el menú lateral (Caja), puedes gestionar todo tu negocio:
*   **Catálogo de Productos:** 
    *   **Gestionar Categorías:** Presiona el botón 📁 para crear, editar o eliminar categorías (Ej: Dulces, Bebidas Alcoholicas). *Nota: No puedes eliminar categorías que tengan productos asignados.*
    *   **Productos:** Crea nuevos productos o edita precios, abreviaturas y si requieren elección de carne.
*   **Diseño de Salón (Croquis):** 
    *   Agrega mesas cuadradas, redondas o rectangulares.
    *   **Cambiar Forma:** Simplemente haz clic en una mesa ya creada para alternar su forma (Cuadrada -> Redonda -> Rectangular).
    *   **Mover:** Arrastra las mesas con el dedo o mouse para acomodarlas según tu local físico.
*   **Disponibilidad de Carnes:** Si se acaba el Pastor, desactívalo aquí y desaparecerá de todas las tablets para que nadie lo pida por error.
*   **Gastos:** Registra compras de verdura, carne o pagos de luz para que tu utilidad neta sea real.
*   **Personal (HRM):** Registra asistencia de meseros y dales adelantos de sueldo.
